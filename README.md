# 🏠 House Price Prediction System

A machine learning-based web application that predicts house prices based on key property features, built with an end-to-end pipeline from data preprocessing to deployment.

 - 🔗 Live Demo: https://vaibhavi49-house-price-prediction-enuxhcf7nlmxdbmkgj9hz3.streamlit.app

---

## 🎯 Objective

To build a real-world regression model that accurately predicts house prices and deploy it as an interactive web application for practical usage.

---

## ⚙️ Tech Stack

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Frontend:** Streamlit  
- **Model Persistence:** Joblib  
- **Deployment:** Streamlit Community Cloud  

---

## 📊 Key Features

- Interactive UI for user input  
- Real-time house price prediction  
- Clean and minimal interface  
- Efficient preprocessing and feature handling  
- Scalable ML pipeline  

---

## 🧠 Machine Learning Approach

- Performed data cleaning and preprocessing  
- Handled missing values and categorical encoding  
- Trained regression models (e.g., Linear Regression / Random Forest)  
- Evaluated model performance using standard metrics  
- Saved and deployed the trained model for inference
